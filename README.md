# PDF表格擷取工具
從PDF檔案中擷取表格

網頁展示: https://ronnywang.github.io/pdf-table-extractor/

命令列工具安裝
--------------------------
1. git clone https://github.com/mozilla/pdf.js.git  
2. cd pdf.js  
3. npm install -g gulp-cli  
4. npm install  
5. gulp generic  
5. cd ../  
6. node parse-cmd.js samples/pta_10229_131308_94274.pdf
